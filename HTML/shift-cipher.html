About shift-cipher


Shift cipher is one of the easiest cryptosystem. It has a 26 possible keys. The Encryption Function is defined as "ek(X) = (X+k) mod 26" and the decryption function is defined as "dk(Y) = (Y-K) mod 26".
This cryptosystem is not secure. As it has only 26 possible keys, it can be broken down using exhaustive search(brute force). The cipher-text attack model can be used to crack the cryptosystem.
